# dotfiles

backup of various scripts, settings, and dotfiles

- `.config`: general config files
- `.rustup`: rust toolchain settings
- `linux`: linux settings and scripts
- `npm`: npm toolchain settings and global packages
- `script`: generic scripts for linux/windows/WSL
- `vim`: vim settings and custom vimscripts
- `vscode`: vscode settings, tasks, and scripts
- `windows`: windows scripts
- `wsl`: WSL settings and scripts
